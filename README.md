Тестовое задание для https://quickresto.ru/

## Цель проекта

Написать мини-аналог google таблиц. На веб странице отображать таблицу 4 на 4 колонки. В клетке можно написать цифру, либо формулу, первым символом которой будет "=".
В формулах возможно ссылаться на клетку, по ее позиции столбец+строка.

Доступные операции: + - / * ( )

После нажатия на enter - производить расфокус клетки и расчет значений таблицы.
Рассчитывать значения нужно на серверной стороне.

## Используемые технологии и инструменты
- Язык программирования: Java
- Фреймворк: Spring Boot
- Сторонние библиотеки: exp4j
